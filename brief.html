ปัญหาหลักที่ต้องแก้

  • SEO — title generic, meta description หาย, schema ไม่มี
  • Commerce — ไม่มี product discovery path ชัดเจน
  • Navigation — พาผู้ใช้ไป content ได้ แต่ยังไม่พาไป shopping intent
  • Conversion — CTA ส่วนใหญ่เป็น Read more แทน Shop / Explore / Find routine
  • System — ยังไม่มีโมดูลระดับ ecommerce เช่น best sellers, categories, reviews, bundles, filters

Homepage ใหม่ควรมีอะไร

  1. Hero พร้อม Shop now และ Explore brands
  2. Shop by category / concern / brand
  3. Best sellers / new arrivals / campaign modules
  4. Signature brand blocks
  5. Routine finder / concern finder
  6. Science & innovation proof section
  7. Sustainability with metrics and proof
  8. Editorial / Journal block ที่ผูกกับ product intent
  9. Newsletter / loyalty / retention module
  10. Deep footer สำหรับ commerce support

Ecommerce baseline

SEO plan

On-page fixes

  • ตั้ง title ใหม่ให้มี keyword intent จริง
  • ใส่ meta description ที่สื่อ category + region + brand value
  • ให้มี H1 ที่ชัดเพียงหนึ่งตัว
  • จัด H2/H3 เป็น keyword clusters
  • เพิ่ม internal linking ไป category, concern, brand, editorial hubs

Technical & structured data

  • Organization
  • WebSite + SearchAction
  • BreadcrumbList
  • CollectionPage / ItemList
  • Product / Offer / AggregateRating
  • Article / NewsArticle
  • XML sitemap, robots, hreflang, canonical discipline
