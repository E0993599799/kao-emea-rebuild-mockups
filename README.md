## 2) Current-page strengths

What is already usable:
- Clean corporate information architecture
- Clear top-level pillars
- Good base content for purpose, innovation, sustainability, careers
- Existing news content can support freshness and SEO
- Brand imagery exists and can be repurposed into richer commerce storytelling

## 3) Current-page gaps blocking a modern ecommerce-ready experience

### UX / commercial gaps
- Homepage is still mostly corporate-information first, not product-discovery first
- No clear shopping journey from hero to category to PDP to cart
- No promotional system: campaigns, bundles, best sellers, seasonal launches, gift guides, skin/hair concern pathways
- No persistent ecommerce CTAs such as `Shop now`, `Find routine`, `Explore collection`, `Subscribe & save`
- Product architecture is implied, not operationalized
- Latest News takes significant space but does not convert to product/category action
- Search appears utility-only, not commerce-grade with predictive intent
- Footer is too light for a modern multi-brand commerce destination

### Content / architecture gaps
- Homepage sections are broad and editorial, but not organized around buyer intent
- No explicit product taxonomy visible from homepage
- No “shop by concern / shop by category / shop by brand / shop by benefit” layers
- No trust system near commerce actions: shipping, returns, subscription, awards, reviews, dermatologist-tested claims, sustainability proofs, certifications

### SEO gaps observed directly
- Empty meta description
- No JSON-LD detected on the homepage
- Heading structure is weak for search intent:
  - browser DOM showed no usable H1 in rendered audit output
  - raw HTML contained 2 H1 tags but rendered hierarchy surfaces the hero statement as H3 and main sections as H2
- Title is generic and under-optimized (`Kao EMEA Homepage`)
- Homepage messaging is brand-purpose heavy and keyword-light for commercial discovery
- Internal linking is broad but not search-strategic for product clusters
- News freshness exists, but commerce landing intent is not prominent

## 4) Rebuild direction

Rebuild the homepage as a hybrid:
- premium corporate brand presence
- modern multi-brand beauty/personal-care commerce front door
- SEO-ready content hub

Recommended posture:
- keep Kao credibility, sustainability, science, and global trust
- move the homepage from “corporate brochure” to “brand-commerce operating system”

### C. Shop entry grid

### D. Signature brand modules
For each priority brand:
- visual brand tile
- short positioning line

- campaign landing page link

### E. Best sellers / trending products
Commerce block should support:
- product cards
- ratings / review count
- price / compare-at price
- quick add

### F. Routine / concern finder
Examples:

### J. Email / loyalty / retention block

### K. Deep footer

## 7) SEO rebuild plan

### Homepage SEO
Replace current generic homepage signals with:
- optimized title
- real meta description
- one clear H1
- supporting H2 clusters aligned to keyword families
- schema markup
- stronger internal linking to category and brand hubs

### Recommended homepage keyword intent clusters
- Japanese beauty brands Europe / EMEA
- hair care products
- skin care products
- sun care / SPF products
- personal care / beauty brands
- sustainable beauty / science-led beauty

### Suggested title direction
Examples:
- Kao EMEA | Hair Care, Skin Care, Beauty Brands & Innovation
- Kao EMEA | Premium Hair, Skin & Personal Care Brands in Europe, Middle East and Africa

### Suggested meta description direction
Example:
- Discover Kao EMEA’s hair care, skin care and personal care brands, science-led innovation, sustainability initiatives, and expert beauty solutions across Europe, the Middle East and Africa.

### Structured data to add
- Organization
- WebSite with SearchAction
- BreadcrumbList
- CollectionPage / ItemList for category pages
- Product, Offer, AggregateRating on PDPs
- Article / NewsArticle for editorial/news pages
- FAQPage where relevant

### Technical SEO baseline
- Core Web Vitals budget
- responsive images / next-gen formats
- semantic HTML
- canonical discipline
- XML sitemap
- robots rules
- hreflang / locale strategy where multiple markets/languages exist
- clean URL taxonomy
- internal search pages noindex as needed

### Content SEO strategy
Build cluster pages, not only campaign pages.
Recommended cluster families:
- shop by hair concern
- shop by skin concern
- ingredient education
- routine pages
- sustainability proof pages
- brand landing pages
- editorial how-to content linked to products
